# ShopHeaven
Online shop with React.js, ASP.NET Core and Entity Framework Core.

Used technologies: 
<ul>
 <li>ReactJS</li>
 <li>MaterialUI</li>
 <li>ASP.NET Core</li>
 <li>Entity Framework Core</li>
 <li>MS SQL Server</li>
 <li>HTML</li>
 <li>CSS</li>
 <li>Azure Storage</li>
 <li>Stripe Payment System API</li>
 <li>JWT and Refresh Token</li>
</ul>
