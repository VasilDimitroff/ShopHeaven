# ShopHeaven
Online shop with React.js, ASP.NET Core and Entity Framework Core. Custom made design, fully responsive.

Used technologies: 
<ul>
 <li>React.js</li>
 <li>MaterialUI</li>
 <li>ASP.NET Core</li>
 <li>Entity Framework Core</li>
 <li>MS SQL Server</li>
 <li>HTML</li>
 <li>CSS</li>
 <li>Azure Storage</li>
 <li>Stripe Payment System API</li>
 <li>JWT and Refresh Token</li>
</ul>
